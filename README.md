# Hawkeyes
Hawkeyes is an eary to use port scanner,

## [`Usage`](docs / Help.md)
`hawk[tool][options]`
## Example
`hawk - t 127.0.0.53 - p 53 -i lo`

## How to compile
1. Download necessary libraries(read [`requirements`](docs/Requirements.md))
2. Run `make` in the project root folder
3. Done !
